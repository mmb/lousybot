IRC bot written in node.js.

* supports plugins
* plugins can use CouchDB for storage
* HTTP and AMQP interface plugins for sending messages to users and channel from scripts
* easy to write plugins that wrap forked child processes to use existing command-line tools and scripts

This bot has been developed on a private server with non-malicious users. Use
it in the wild on IRC at your own risk.
